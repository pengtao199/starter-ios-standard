# Plans Index

`Plans/` 用来放阶段性方案、执行计划和决策记录。

文档边界：

1. `README.md` 是 Starter 使用说明
2. `AGENTS.md` 和 `Rules/` 是长期开发规则
3. `Docs/` 是项目说明、PRD 和参考材料
4. `Plans/` 是阶段推进记录

## 命名规范

计划文档使用两位序号开头：

```text
00-XXXX.md
01-XXXX.md
02-XXXX.md
```

规则：

1. 序号表示计划创建顺序，不表示优先级
2. `XXXX` 使用英文大写短横线命名
3. 一个计划只覆盖一个明确主题
4. 计划完成后保留文档，用于追踪基座演进历史
